init(subject:reducer:middleware:emitsValue:)

Required initializer that configures the action handler pipeline and the state storage

Declaration

init<M>(subject: UnfailableReplayLastSubjectType<StateType>, reducer: Reducer<ActionType, StateType>, middleware: M, emitsValue: ShouldEmitValue<StateType> = .always) where ActionType == M.InputActionType, StateType == M.StateType, M : MiddlewareProtocol, M.InputActionType == M.OutputActionType

Parameters

subject

a reactive subject type that replays the last value, never fails and works on StateType elements. It should contain the initial state already.

reducer

a reducer function wrapped in a monoid container of type Reducer, able to handle the state of the type StateType and actions of type ActionType. For reducer composition, please use the diamond operator (<>) and for reducers that understand only a sub-state part, use the lift functions to elevate them to the same global state and global action type.
