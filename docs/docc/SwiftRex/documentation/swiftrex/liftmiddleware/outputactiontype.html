LiftMiddleware.OutputActionType

The Action type that this MiddlewareProtocol will eventually trigger back to the store in response of side-effects. This can be the same as LiftMiddleware.InputActionType or different, in case you want to separate your enum in requests and responses.

Declaration

typealias OutputActionType = GlobalOutputActionType

Discussion

Most of the times middlewares donâ€™t need to dispatch all possible actions of the whole global action tree, so we can decide to allow it to dispatch only a subset of the action, or not dispatch any action at all, so the LiftMiddleware.OutputActionType can safely be set to Never.

In this case, this action type can be a subset to be lifted to a global action type in order to compose with other middlewares acting on the global action of an app. Please check doc:Lifting for more details.
