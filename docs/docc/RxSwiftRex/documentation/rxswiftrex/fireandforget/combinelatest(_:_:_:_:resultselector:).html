combineLatest(_:_:_:_:resultSelector:)

Merges the specified observable sequences into one observable sequence by using the selector function whenever any of the observable sequences produces an element.

Declaration

static func combineLatest<O1, O2, O3, O4>(_ source1: O1, _ source2: O2, _ source3: O3, _ source4: O4, resultSelector: @escaping (O1.Element, O2.Element, O3.Element, O4.Element) throws -> Self.Element) -> Observable<Self.Element> where O1 : ObservableType, O2 : ObservableType, O3 : ObservableType, O4 : ObservableType

Return Value

An observable sequence containing the result of combining elements of the sources using the specified result selector function.

Parameters

resultSelector

Function to invoke whenever any of the sources produces an element.
