Type Method

zip(_:resultSelector:)

Merges the specified observable sequences into one observable sequence by using the selector function whenever all of the observable sequences have produced an element at a corresponding index.

Declaration

static func zip<Collection>(_ collection: Collection, resultSelector: @escaping ([Collection.Element.Element]) throws -> Self.Element) -> Observable<Self.Element> where Collection : Collection, Collection.Element : ObservableType

Return Value

An observable sequence containing the result of combining elements of the sources using the specified result selector function.

Parameters

resultSelector

Function to invoke for each series of elements at corresponding indexes in the sources.
