Instance Method

flatMap(_:)

Projects each element of an observable sequence to an observable sequence and merges the resulting observable sequences into one observable sequence.

Declaration

func flatMap<Source>(_ selector: @escaping (Self.Element) throws -> Source) -> Observable<Source.Element> where Source : ObservableConvertibleType

Return Value

An observable sequence whose elements are the result of invoking the one-to-many transform function on each element of the input sequence.

Parameters

selector

A transform function to apply to each element.
