Instance Method

multicast(_:)

Multicasts the source sequence notifications through the specified subject to the resulting connectable observable.

Declaration

func multicast<Subject>(_ subject: Subject) -> ConnectableObservable<Subject.Element> where Subject : SubjectType, Self.Element == Subject.Observer.Element

Return Value

A connectable observable sequence that upon connection causes the source sequence to push results into the specified subject.

Parameters

subject

Subject to push source elements into.

Discussion

Upon connection of the connectable observable, the subject is subscribed to the source exactly one, and messages are forwarded to the observers registered with the connectable observable.

For specializations with fixed subject types, see publish and replay.
